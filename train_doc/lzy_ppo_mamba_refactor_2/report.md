# 一、总体结论（核心判断）

当前训练状态可归纳为：

> **策略已“过度收敛到局部行为”，探索不足，学习信号趋弱，处于“伪稳定状态”。**

具体表现为：

* PPO loss 快速塌缩 → 接近 0
* clip_frac 长期维持高位（≈0.7–0.9）
* KL 先高后低 → 当前偏低
* entropy 低且稳定
* reward 无明显上升趋势
* 行为模式单一（高 move_rate + 中等 still_rate + 低 flash 成功）

这不是“收敛成功”，而是：

> **策略被 clip 限制住 + advantage 信号变弱 → 更新停滞**

# 二、训练动态分析

## 1. PPO优化状态

### 现象

* `total_loss / policy_loss / value_loss` → 快速下降到接近 0
* `entropy_loss` → 稳定在低值（≈ -2）

### 解释

* 优势函数 A_t 接近 0（或被 normalize 压平）
* policy 更新幅度极小
* value function 已拟合当前（次优）策略

👉 结论：

> **梯度信号已经衰减，模型进入“低学习区”**

---

## 2. clip机制（关键问题）

### 现象

* `clip_frac ≈ 0.7–0.9`（长期偏高）

### 含义

\text{clip_frac} = \mathbb{E}[\mathbf{1}(|r_t - 1| > \epsilon)]

* 大量样本被 clip
* 实际优化目标变成：

  ```
  min(ratio * A, clipped_ratio * A)
  ```

  → 梯度被截断

### 结论

> **策略更新被 clip 强限制 → 学习“撞墙”**

---

## 3. KL 散度

### 现象

* 初期 KL ≈ 1.0+
* 后期降至 ≈ 0.2–0.4

### 解释

* 初期更新过猛
* 后期进入保守更新区

👉 问题：

> KL 偏低 + clip_frac 偏高 = **更新方向被限制但仍频繁触发clip**

---

## 4. 梯度情况

### 现象

* grad_norm 从 60 → <5 → ≈0

### 解释

* 梯度消失趋势明显
* loss → 0 导致反向传播无信号

# 六、调参建议（优先级排序）

## 【高优先级】

### 1. 降低 clip 强度

```python
clip_range = 0.2 → 0.1 或 0.05
```

作用：

* 减少 clip_frac
* 允许更细粒度更新

---

### 2. 提高 entropy（恢复探索）

```python
c2 (entropy coef) = 0.01 → 0.02~0.05
```

或：

```python
entropy annealing（反向退火）
```

---

### 3. 增强 reward 信号

建议加入：

```text
+ 生存奖励（每 step 正奖励）
+ 探索奖励（新格子）
+ 距离怪物惩罚（连续）
```

避免：

* 稀疏奖励（现在就是问题）

---

## 【中优先级】

### 4. 降低学习率

如果 early KL 很高：

```python
lr = 3e-4 → 1e-4
```

---

### 5. 增大 batch / rollout

```python
T 增大（更稳定 advantage）
```

---

### 6. 调整 GAE

```python
lambda = 0.95 → 0.97~0.99
```

增强长期 credit

---

## 【结构优化】

### 7. 动作空间问题（重要）

当前：

> flash 几乎没学会

建议：

* 分离 action head：

  * move
  * skill（flash）

或：

* 强制使用：

  ```text
  每 N 步必须 flash（探索阶段）
  ```

---

### 8. 加 auxiliary loss（你已有但太弱）

* 位置预测 ✔
* 怪物距离 ✔

建议：

```python
aux_loss weight ↑
```
