# Project Documentation

This directory keeps project-owned documentation for the PPO agent and related
local notes.

## Local Algorithm Docs

- `ppo/README.md`: current PPO + Mamba design documentation.
- `agent_ppo_adv1/README.md`: older `ppo_adv1` local algorithm notes.
- `agent_diy/README.md`: local migrated `diy` algorithm notes.

## Excluded Local References

The following local-only directories are intentionally ignored for public
source releases:

- `html/`: exported Tencent Arena documentation pages.
- `md/`: converted Tencent Arena documentation pages.

Keep those files local or share them only through a private channel when their
copyright, access scope, or experiment details are acceptable to disclose.
